caramelize

Caramelize is a compact and flexible wiki converter. It is intended for easily creating export of otherwise rare supported legacy wikis. With caramelize you can create your own export configurations and transfer your data into a git-based gollum-wiki retaing all your history.

scraperrr.js

By defining JSON-configurations for Scraperrr.js, textual website contents can be targeted, read and exported in a specified JSON-format. The software was specifically designed for reading and converting structured information from Wikis, but can be applied more generally.

  • Github

ImageFlow

ImageFlow was my Bachelor-project and is the implementation of a node-based user interface for ImageJ. This graphics software gives you a workspace for connecting graphics filters to create a processing workflow. This is an easy alternative to the script-based Macro-interface of ImageJ.

  • Github

DDS-Utils

The DDS-Utils are a number of small graphic applications for special use-cases in the games-content-production process. Badds allows to batch process many texture files at once, Dropps is an easy tool for texture conversion. Radds is a for review and previewing textures. Madds is for displaying memory requirements in a treemap.
